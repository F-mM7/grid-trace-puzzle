# Grid Trace Puzzle

## 概要
2つの端点（青い丸）を一筆書きで結ぶパスパズルゲームです。グリッド上のすべての対象セルを通る1本の道を見つけてください。

## デモ
[Grid Trace Puzzle](https://futa.github.io/grid-trace-puzzle/)

## 機能
- 4×4〜12×12のグリッドサイズに対応
- 唯一解が保証されたパズルの自動生成
- ドラッグで線を描画、クリックで線を削除
- リアルタイムの解答チェックと成功アニメーション
- タッチデバイス対応

## 技術スタック
- React 19
- TypeScript
- Vite

## セットアップ
```bash
git clone https://github.com/futa/grid-trace-puzzle.git
cd grid-trace-puzzle
npm install
npm run dev
```

## ビルド
```bash
npm run build
```
