# Grid Trace Puzzle

## 概要
数字を線でなぞって囲むロジックパズルゲームです。グリッド上に配置された数字に対応する数のセルを線でつなげて囲みます。

## デモ
[Grid Trace Puzzle](https://futa.github.io/grid-trace-puzzle/)

## 機能
- 複数のグリッドサイズ（3x3〜15x15）に対応
- 自動生成される唯一解パズル
- ドラッグ＆クリックによる直感的な線の描画
- 解答チェックと成功アニメーション
- レスポンシブデザイン

## 技術スタック
- React 18
- TypeScript
- Vite
- CSS3 アニメーション

## セットアップ
```bash
git clone https://github.com/futa/grid-trace-puzzle.git
cd grid-trace-puzzle
npm install
npm run dev
```

## ビルド
```bash
npm run build
```
